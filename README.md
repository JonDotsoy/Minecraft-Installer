Minecraft Installer
===================

Un archivo shell que permite instalar correctamente Minecraft para linux

Requisitos:
 - notify-send (>= 0.7.5)
 - java (>= 1.7.0)
 - wget (>= 1.13.4)
 - gnu (>= 3)
